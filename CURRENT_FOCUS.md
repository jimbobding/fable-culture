# 🔥 Current Caribbean Priorities

## 1️⃣ Reusable Cultural Spotlight System

A reusable “Cultural Spotlight” system is planned.

Purpose:

- highlight a major cultural identity element
- create memorable emotional anchors
- deepen storytelling

Examples:

- Carnival
- Reggae
- Steelpan
- Spice culture
- Rastafari
- Windrush
- Cricket culture

---

### Important Architecture Rule

The spotlight system should be:

👉 CONDITIONAL

NOT every country requires one.

If a country does not have:

- a strong spotlight theme
- a major defining cultural identity section

then:

- the page should still work perfectly
- layouts should not break
- spotlight section should simply not render

---

### UX Goals

The spotlight system should:

- feel immersive
- support layered visuals
- support editorial storytelling
- break up repetitive layouts

NOT:

- feel like another generic card section
